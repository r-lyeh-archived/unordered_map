=============

Portable header for `std::unordered_map<K,V>` template. Welcome to C++. Enjoy!

Usage
-----

```c++
#include "unordered_map.hpp"

int main( int argc, const char **argv ) {
    std::unordered_map<int,double> umap;
    return 0;
}
```
